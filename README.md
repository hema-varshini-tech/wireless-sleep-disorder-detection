# Wireless Sleep Disorder Detection Using IoT

## Overview
This project focuses on developing an IoT-based healthcare monitoring system for detecting sleep-related disorders using wearable sensor technology and wireless communication. The system is designed to monitor important physiological parameters during sleep and assist in identifying abnormal sleep conditions.

The project combines Biomedical Engineering concepts with IoT, embedded systems, and real-time health monitoring technologies.

---

## Project Status
🚧 Currently Under Development

This project is actively being developed and improved with additional sensor integration, real-time monitoring capabilities, and data analysis features.

---

## Objective
The main objective of this project is to create a smart wireless healthcare monitoring system capable of tracking sleep-related parameters and supporting early identification of possible sleep disorders.

## Working Principle
The system collects physiological data from sensors connected to the ESP32 microcontroller. The monitored parameters are processed and transmitted wirelessly for analysis. Based on the collected data, abnormal sleep patterns can be identified for healthcare monitoring purposes.
